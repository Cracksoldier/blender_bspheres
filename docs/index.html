bSpheres NX

zSpheres-style base-mesh creation for Blender.

Blender 4.2 LTS → 5.x
Download latest release View on GitHub

What it is

zBrush has a handy tool called zSpheres for blocking out quick base meshes. Blender can do the same thing with a single-vertex mesh and the Mirror, Skin, and Subdivision Surface modifiers — but the setup is fiddly. bSpheres NX wraps all of that behind one panel so you can just start sketching, then bake the result into a watertight, sculptable mesh.

Install

  1. Download bspheres-<version>.zip from the Releases page.
  2. In Blender, open Edit → Preferences → Get Extensions.
  3. Use the dropdown (top-right) → Install from Disk… and pick the zip.
  4. Enable bSpheres NX, then press N in the 3D Viewport and open the bSpheres NX tab in the sidebar.

Quick guide — the panel

Create

Builds the single-vertex mesh with the Mirror, Skin, and Subdivision modifiers already set up.

Mirror Axis X / Y / Z

Toggle symmetry on each axis while you work.

Mark Root

Set the selected vertex as the skin root — handy when building bones from the vertices.

Mark / Clear Loose

Make vertices act like a grab brush, pulling the skin toward them instead of centering it.

Viewport

How much to subdivide the skin. Higher = smoother preview.

bSkin Settings

Per-object output controls: voxel remesh toggle + size, shade-smooth, merge doubles + threshold, recalculate normals, and thin-branch warning + min radius. Applied by Make bSkin, Preview, and Apply.

Selected Node

Edit Mode only. Shows the active vertex's live skin radius and root/loose flags. Mark / Clear Preserve adds or removes the vertex from the thin-branch warning exemption. Select Children / Select Parents selects all downstream or upstream vertices from the active joint — useful for posing a full limb as a unit.

Generate Armature

Converts the bSphere control mesh into a Blender armature: each edge becomes one bone, parented to follow the vertex graph from the skin root. Output goes into a bSpheres_Armatures collection. Only the unmirrored half is included — add an Armature Mirror modifier afterwards if needed.

Preview / Refresh

On-demand preview in a bSpheres_Preview collection. Re-clicking updates the same object in-place. Use Delete to remove it.

Make bSkin

Non-destructive permanent bake. Creates a new mesh in bSpheres_Output without touching the control structure — iterate as many times as you like.

Apply

Destructive bake. Applies the three modifiers onto the control object, then post-processes using the same bSkin Settings. Use when done iterating.

Editing shortcuts

EExtrude vertices to grow the shape.
Ctrl + AScale the skin at the selected vertex (like scaling a zSphere).
Ctrl + RSelect two vertices, then click between them to add a vertex.
GMove vertices freely from any view.
